What makes it different

◈

Zero Configuration

Open the app on two devices. They find each other automatically via mDNS — no IP addresses, no pairing codes, no accounts.

⤵

Any File. Any Size.

Raw byte streaming with 4MB chunked transfer and SHA-256 integrity verification per chunk. Videos, archives, ISOs — no restrictions.

⬡

Text Transfer

Send raw text between devices instantly. No clipboard sync tool. No third-party service. Just type and send.

↻

Resumable Transfers

Network dropped mid-transfer? Each chunk is independently verified and tracked. Resume from where you left off.

◑

Multiple Themes

Seven themes across dark and light palettes. Void, Abyss, Slate, Cosmos, Rose, Citrus, Sky. Switch instantly.

⊞

Native Desktop App

Built with Tauri. A real native window — not a browser tab. Drag files straight from Explorer. No runtime required after install.

How it works

01

Download and install The Rift on both machines. Runs on Windows, macOS, Linux.

02

Connect both devices to the same WiFi network — or create a hotspot from one to the other.

03

Open The Rift. Both devices appear automatically in the left panel within seconds.

04

Select a device, drop your files, click Send. Files land in Downloads on the other side.
