Capabilities

What makes it different

⚡

Multi-Stream Transfer Engine

4 parallel TCP worker streams per file, pipelined dispatch. BLAKE3 integrity verification on every chunk. An independent overseer cross-checks progress against the manifest in real time with live speed and ETA on both ends.

New in v1.1.1
⬡

Triple-Layer Discovery

mDNS for zero-config local resolution, UDP broadcast for cross-subnet reach, active subnet scan as fallback. Batch discovery events prevent UI frame drops at scan completion. Devices appear within seconds.

New in v1.1.1
◉

Android Native

Full native mobile layout with three-tab interface — Devices, Send, History. In-app file browser backed by a storage scanner, no system picker required. Three-tier fallback for maximum OEM compatibility. Android 7.0+.

New in v1.1.1
⊞

Windows Hotspot

Automated hotspot creation via WinRT with netsh fallback for older adapters. Detects an already-running hotspot without duplicating it. Join another device's hotspot directly from inside the app.

New in v1.1.1
↻

Connection Stability

Persistent TCP rift channel between transfers with bond-keeper pings every second — no radio idle eviction. Heartbeat monitoring with automatic device eviction after sustained failure. ~11-second dead-peer detection.

New in v1.1.1
⤵

Any File. Any Size.

Raw byte streaming with 4 MB chunked transfer. Videos, archives, ISOs, APKs — no restrictions. Resumable from the last verified chunk if a transfer is interrupted mid-stream.

◈

Zero Configuration

Open the app on two devices and they find each other. No IP addresses, no pairing codes, no accounts, no cloud relay, no setup wizard. It simply works.

◑

8 Themes

Void, Abyss, Slate, Cosmos, Rose, Citrus, Sky, and Auto — which follows your system preference. Dark and light palettes. Switch instantly, no restart required.

Process

How it works

01

Install The Rift on both machines. Windows x64 EXE or MSI installer. Android APK via sideload — no Play Store needed. No account, no sign-up.

02

Connect both devices to the same WiFi — or use the built-in Windows hotspot tool to create one right from inside the app.

03

Open The Rift on both devices. Triple-layer auto-discovery finds them within seconds. No IP addresses, no pairing codes.

04

Select a device, drop your files, click Send. The recipient accepts and 4-stream transfer begins immediately. Real-time speed and ETA on both sides.
